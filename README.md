# baus-ceo-web 🌐

Repositorio **público** del ecosistema de marca BAUS. Contiene únicamente los activos aprobados para visualización externa.

## Regla de Oro
> **Antigravity-All** = Cerebro privado → nunca a internet  
> **baus-ceo-web** = Vitrina pública → solo lo que Fausto aprueba

## Estructura
```
/                           → baus.ceo (homepage)
/sonar-hall/propuestas/     → propuestas comerciales del venue
/entertainment/             → división Global Entertainment
/media-tech/                → división Media & Technology
/advisory/                  → división Advisory & Consulting
```

## Cómo agregar una propuesta
1. Generar el HTML en Antigravity-All (repositorio privado)
2. Aprobar el contenido con Fausto
3. Copiar **solo** el HTML aprobado a este repo
4. Commit + push → Hostinger despliega automáticamente
