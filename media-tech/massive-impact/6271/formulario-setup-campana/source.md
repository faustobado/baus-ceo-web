# Formulario RFP de Campaña (Request for Proposal)

> **Documento fuente editable.** Este archivo contiene la estructura y documentación del formulario de configuración de campañas de Massive Impact.

## Descripción del Formulario

Este formulario está diseñado para que el equipo de **Massive Impact** e integradores comerciales recopilen toda la información necesaria para configurar el setup de una campaña nueva y obtener los requerimientos del cliente.

El formulario está dividido en 6 secciones principales:
1. **Información General:** Datos del AE (ejecutivo de cuenta), líder de ventas, campaña, agencia y marca.
2. **Datos Básicos de la Campaña:** Fechas, presupuesto, creatividades disponibles y KPIs de canal.
3. **Targeting de la Campaña:** Segmentación geográfica y de audiencias.
4. **Estrategia de Campaña:** Canales a incluir, presupuestos por canal y exclusiones.
5. **KPIs y Medición de Éxito:** Criterios para renovación y KPI primario por canal (CTV, OTT, etc.).
6. **Información Adicional:** Archivos adjuntos y notas finales.

El envío de los datos se procesa de forma asíncrona hacia una Web App de Google Apps Script que almacena las solicitudes en una hoja de cálculo unificada.
